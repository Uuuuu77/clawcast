*"See Tomorrow, Today"*

A dark, futuristic prediction agent that analyzes global events across sports, crypto, politics, finance, technology, culture, and music using AI-powered web research.

---

## **Core Features**

### 1. **Futuristic Dashboard**
- Dark theme with neon cyan/purple accents and glowing effects
- **Prediction Cards** showing trending events with probability meters
- Category filters (Sports, Crypto, Politics, Tech/AI, Finance, Culture, Music)
- Visual probability indicators with animated confidence gauges

### 2. **Oracle Chat Assistant**
- Conversational AI sidebar for deep-dive analysis
- Ask questions like *"Who will win the Super Bowl?"* or *"Will Bitcoin hit $100K this year?"*
- Provides reasoning, sources, and confidence levels
- Web search integration for real-time information gathering

### 3. **Prediction Explorer**
- Search bar to query any upcoming event
- AI returns structured predictions with:
  - **Probability percentage** (visual meter)
  - **Key factors** supporting the prediction
  - **Sources** from web research
  - **Risk assessment**

### 4. **My Predictions History**
- Track all your queried predictions
- See if predictions were accurate after events conclude
- Personal accuracy score over time
- Filter by category and date

### 5. **Trending Events Feed**
- Auto-curated list of upcoming major events
- Categories: Championships, Elections, Crypto milestones, Tech announcements, Awards shows
- Quick-tap to get AI analysis
